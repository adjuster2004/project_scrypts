# project_scrypts
Project scrypts помогают структурировать папки в проектах, создавать и выводить структуру в TXT формате

## Генерация структуры проекта
- generate_structure.py

Создает нужную структуру папок и файлов. Структура описывается в функции **create_project_structure**

- generate_tree.py

Наборот, пробегает по папкам и выводит структуру папок-файлов в TXT. Не забудьте положить py файл в корень проекта.

- tree_view.py

Тоже самое, что и generate_tree.py, но при запуске попросит ввести путь к корню проекта.

- test_smtp.py

Создает SMTP подключение на указанный сервер и порт, сообщает серверу, что во вложении есть файл EXE. Для проверки фильтра вложений.
